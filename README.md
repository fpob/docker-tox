# tox

Image for running [tox](https://tox.readthedocs.org/en/latest/) with multiple
Python and PyPy versions.

* Python 3.6
* Python 3.7
* Python 3.8
* Python 3.9
* Python 3.10
* PyPy 3.7
* PyPy 3.8


## Example usage

```sh
docker run -v $PWD:/workdir registry.gitlab.com/fpob-dev/docker/tox tox
```
